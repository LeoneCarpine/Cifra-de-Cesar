# Criptografia de Arquivo Texto em C

Este projeto foi desenvolvido como atividade da disciplina de Redes/Segurança, com base no material de aula sobre criptografia. 
O programa realiza a leitura de um arquivo texto e aplica a técnica da **Cifra de César**, uma cifra de substituição clássica 
em que cada letra do texto é deslocada por uma quantidade fixa de posições no alfabeto.

## Funcionalidades

O programa possui um menu interativo com as seguintes opções:

1. Ler arquivo texto  
2. Criptografar o conteúdo  
3. Descriptografar o conteúdo  
4. Salvar o arquivo  
5. Sair  

A técnica utilizada foi a **Cifra de César com deslocamento k = 3**, onde, por exemplo, 
a letra `A` é transformada em `D`, `B` em `E`, e assim por diante. Para descriptografar, o processo inverso é realizado.

## Objetivo

O objetivo do projeto é demonstrar, de forma prática, o funcionamento básico de uma técnica de criptografia por substituição, 
permitindo ao usuário proteger e recuperar o conteúdo de arquivos texto.
